June 11, 2026 — Configuration Standardization & Portal CI/CD

Resolved cross-environment configuration inconsistencies and unified all portal CI/CD workflows.

Key Achievements

  • Quotation API Path Standardized: Fixed api/quotations (plural) → api/quotation (singular) across ALL layers:
    • Kubernetes: 4 claims configmaps (dev/qa/staging/prod)
    • Entity Portal: 4 environment files × 4 branches
    • Terraform: Commented module reference

    • Single workflow file supports all 4 environments
    • Guardrails validate branch-to-environment mapping
    • GitHub Environments configured with WEBAPP_NAME, REGISTRY, ACR_NAME
  • All 6 portal pipelines passing (admin dev/qa/staging, entity dev/qa/staging)
